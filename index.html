How to Add Pictures

  1. Upload your image into the images/ folder on GitHub.
  2. Open photos.js and add one new item to the list.
  3. Commit the change. GitHub Pages will update the site.

This version is static, so it does not do direct browser uploads on the live site. If you want that later, we can add a simple upload backend.
